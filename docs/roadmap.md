# Roadmap — Evolução para Integrações Reais

- Pagamentos
  - Integração com gateway (ex.: Stripe, Mercado Pago, Pagar.me)
  - Fluxo de carteiras locais (Pix, boleto) com webhooks
- Frete/Logística
  - Cálculo de frete real (Melhor Envio, plataformas de logística)
  - Rastreamento real com eventos por transportadora
- Catálogo
  - Importação/integração com ERP/OMS (estoque e preços)
  - Variações com SKU/GTIN e imagens por variação
- Autenticação/Conta
  - Login social (Google/Apple), MFA e redefinição de senha real
  - Endereços via API de CEP (ex.: ViaCEP)
- Admin
  - Upload real de imagens (S3/Storage) e CDN
  - Relatórios avançados e exportações assíncronas
- Observabilidade
  - Logs, métricas e monitoramento de erros (Sentry, etc.)
- SEO/Conteúdo
  - Headless CMS para banners e páginas institucionais
